# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-05-04

### Added
- **Real-time Streaming**: Watch tokens appear live from all models simultaneously.
- **Enhanced UI**: Sophisticated boxed layout with word wrapping and refined terminal styling.
- **Ollama Lifecycle Management**: Automatic installation checks and service auto-start.
- **Deep Metrics**: Detailed performance tracking for IN/OUT tokens, TTFT, and load durations.
- **GitHub Templates**: Standardized Pull Request and Issue templates for contributors.

### Changed
- Refactored CLI execution flow to support both streaming (default) and batch modes.
- Updated documentation with comprehensive feature guides and usage examples.

### Fixed
- Resolved build errors related to UI rendering and output buffers.

---

## [1.0.0] - 2026-04-29

### Added
- Initial CLI for parallel local LLM comparison.
- Basic model discovery and metrics.
